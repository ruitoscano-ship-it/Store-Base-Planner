# Kenney Mini Market 3D models (CC0)

These GLB files are from [Kenney Mini Market](https://kenney.nl/assets/mini-market) (CC0 license).

| File | Used for |
|------|----------|
| `shelf-boxes.glb` | Ambient gondola shelf |
| `freezers-standing.glb` | Cold refrigerated shelf |
| `display-bread.glb` | Hot prepared-food shelf |
| `cash-register.glb` | Checkout till / POS |
| `fence-door-rotate.glb` | Gated entry (turnstile gate) |
| `wall-door-rotate.glb` | Open entry (store door) |

Loaded via Three.js `GLTFLoader` and scaled to match backoffice prop dimensions (width × depth × height in meters).
